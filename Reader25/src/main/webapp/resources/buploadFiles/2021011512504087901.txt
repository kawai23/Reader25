<책방>
1) 토론방
- 책방 : 원하는 책과/ 날짜, 시간을 정하면 누구든지 들어와서 토론하는 방
- 그룹방 : 원하는 그룹과 토론하는 방(아이디 클릭하면 토론에 참여하기, 토론 초대)

2) 책리뷰
- 회원이 올린 책리뷰 >> 마이페이지에서 기록이 있음
- 책 분류에 따라 정렬
- 같은 책을 읽은 여러 리뷰 볼 수 있음! 

3) 책투기방
- 정기적으로 책을 만들어보기
- 완성시 사이트에 책 출판 가능

4) 랭킹
- 분야별 랭킹 
- 리뷰 랭킹 (회원 중 다독왕, 리뷰왕, 토론왕)
- 책의 명대사를 뽑아라!?!

5) 책교환방
- 원하는 책 서로 교환하기


<기존 사이트와 차이점>
1. 디베이팅데이  : 토론 사이트 , 토론을 위주로 한다.
2. 도서관사이트 : 작은 독서토론 만듦, 원격으로 진행이 안됨, 책관리위주
3. 책리뷰사이트 : 각 블로그에 게시해서 서로 보고, 관리하기 어려움
>> 앞으로는 원격으로 많이 만날것!

원격으로 진행되는 독서방

1) 독서토론을 화상으로 진행 : WebRTC api
2) 책리뷰 : 회원들 책 리뷰 게시글 올림 > 같은 책을 읽은 사람들 리스트를 볼 수 있고 토론에 초대 가능
3) 책만들기 : 회원들이 책을 만들어 보는 기회를 올리고 그 책을 읽은 사람들은 후원, 리뷰
4) 예약페이지 : 독서토론이 관리가 가능하게 만들 수 있음, 토론 후 토론일지 작성 >
