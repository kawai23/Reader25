## 프로젝트 개요
코로나19의 여파로 밖에서 독서 모임, 독서 토론을 즐기지 못하는 사람들을 위해 웹적으로나마 독서의 즐거움, 토론의 즐거움을 느낄 수 있게 하고 싶다는 생각으로 인해 구현하게 되었습니다.
‘새로워진 독서’ 독서를 혼자 즐기는 것이 아니라 다양한 방법으로 많은 사람들과 나눌 수 있는 신개념 독서 커뮤니티
